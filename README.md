# calculadora_IA
App de una claculadora Ross-Heidecke

## Prueba rápida

- Abra el archivo `calculadora.html` en el navegador para ejecutar la calculadora localmente.
- Si usa GitHub Codespaces / VS Code: haga clic derecho en `calculadora.html` y seleccione "Open Preview" o use la extensión Live Server.
- Alternativa (terminal): sirva la carpeta y abra la ruta en el navegador:

```bash
python -m http.server 8000
# luego abrir http://localhost:8000/calculadora.html
```

Todos los cálculos se realizan localmente en el navegador (JavaScript). Los campos validan valores incoherentes (VRN > 0, V > e, etc.).
